# Retail-Sales-Performance-Analysis
A data analysis project analyzing regional sales performance and profit margins using Python

### Key Features
* **Data Wrangling**: Cleaning raw data, handling missing values, and mapping localized formats to English standards.
* **Financial Metrics**: Calculating Weighted Profit Margins to assess regional efficiency accurately.
* **Visualization**: Generating dual-axis charts to correlate Sales Volume (Bar) with Profit Margin (Line).

### How to Run
1.  Ensure you have `pandas`, `matplotlib`, and `seaborn` installed.
2.  Run the script:
    ```bash
    python Retail_Analysis.py
    ```

### Dependencies
* Python 3.x
* pandas
* seaborn
* matplotlib
